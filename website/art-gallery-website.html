Art Gallery Website

Created For

"Information Architecture for the Web" course at Rasmussen College â€“ Winter Quarter (2017)

Goal

The goal of this project is to code and design a fictitious art gallery website that is organized and has clear navigation. Both form and function are used to create a website that helps users easily navigate and accomplish their goals.

Materials Used

  • Adobe Dreamweaver
  • Adobe Photoshop
  • Adobe Illustrator
  • Pexels for stock photography
  • Veodesign for the social media icons
  • Mikhail Niedre at CodePen for the modal gallery and carousel code

Introduction

Once I received this project's requirements, I began thinking about the website's navigation. I thought about everything from how many navigation levels to include to what to name each navigation and menu option. My goals were to create clear labels; convey information when and where users expected it; and create an engaging, usable, and useful website.

Assets

I created navigation hierarchy and structure diagrams. These diagrams helped me organize my thoughts and determine the best locations for the content. Thinking through the navigation this way ensured that the navigation helped users accomplish their goals and find the information they needed.

Final Result

After I created the logo and written content and found stock photography, I began building the website. I focused on making each page clean, clear, and usable. I also made sure that the website's content and look were appropriate for an art gallery and portrayed Altered Visions in a professional manner.

This project taught me how important information architecture is for web design projects, and how design, content, and structure are required for websites to be effective.
