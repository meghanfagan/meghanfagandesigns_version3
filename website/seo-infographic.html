Created For

"Search Engines, Optimization, and Analytics" course at Rasmussen College â€“ Winter Quarter (2017)

Goal

The goal of this project is to use color, graphics, and text to create an engaging infographic about search engine optimization (SEO).

Materials Used

  • Adobe Illustrator

Introduction

SEO is a powerful tool anyone can use to make websites easier to find, understand, and use. I made it my goal to show through this infographic how effective and easy to implement SEO techniques are.

Final Result

To make my infographic eye-catching and easy to understand, I created fun and simple vector graphics that clearly conveyed the concepts described in the text. Readability is important with infographics, so I used negative space to give each concept room to breathe, and teal boxes to clearly number and differentiate each concept.
