Academic Symposium Poster

Created For

Rasmussen College's 2016 Academic Symposium poster and logo design contest.

Goal

The goal of this project is to create a logo design for Rasmussen College’s 2016 Academic Symposium that can be used on posters, banners, and so on.

Materials Used

  • Adobe Illustrator

Introduction

I first reviewed the technical specifications for this project, which noted the Symposium’s theme, color scheme, and typefaces. I then went to work researching design thinking and imagery.

Final Result

After creating sketches, I used the Pen, Shape, and Pathfinder tools in Adobe Illustrator to create the vector icons and illustrations. My goal was for the poster to look interesting and fun, yet professional.

Although my poster was not selected, this project was a great learning experience – it taught me how to create a design using technical specifications from a client, how to meet a deadline on short notice, and how to create unique and clear vector illustrations.
